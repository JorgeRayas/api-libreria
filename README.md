# api-libreria

Proyecto de libreria operaciones básicas de manejo de datos

## Operaciones
- AltaLibro
- ObtenerLibros
- CambioLibro
- EliminarLibro
- ObtenerCatalogos

## Instalación
Para poder poner en marcha el servicio se deben configurar las variables de ambiente siguientes:
```
- BD_SERVER
- BD_NOMBRE
- BD_USUARIO
- BD_CONTRASENA
```
